# Cargo Website — Typhex Webster

Static website prototype built with Claude Design.

## Tech Stack
- HTML / CSS / JavaScript
- React 18 (via CDN)
- Babel Standalone (in-browser JSX transformation)
- No build step required

## Local Development
Just open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deployment
Deployed via Vercel (auto-deploys on push to `main`).
